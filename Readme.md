# BarCopa

## Descripción
Se desarrolló una página estática para el Bar Copa, situado en la calle Alcalá.  
La página contiene una cabecera con su logo y un enlace que redirecciona a las distintas secciones.  
También incluye un contenido principal donde se muestra una breve reseña del bar, una sección secundaria con sus servicios, un formulario de contacto y una encuesta.
## Tecnología

Proyecto de una página web estática creada con **HTML y CSS**.
